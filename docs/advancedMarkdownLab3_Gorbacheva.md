## Расширенный Markdown: доподнительные возможности

**Чекбоксы**
- [] Первая задача[^1]
- [x] Вторая задача
- [] Третья задача

**Сноска**

[^1]: Задача еще не выполнена!

**Alert-блок**
>[!TIP]
>**Совет по использованию:** Для экранирования специальных символов в Markdown используйте обратный слеш.

**Escape-символы**

- Заголовки без форматирования: `\# Это не заголовок`
- Звездочка как текст: `\*курсив\*`

**HTML в Markdown**

```HTML
<p style="background-color: #8ff9fa; padding: 15 px; border-radius: 5px;">
    Этот параграф оформлен с помощью <strong>HTML-тега</strong> внутри Markdown-документа.
</p>
```
