Markdown - это простой язык разметки для быстрого форматирования текста (заголовки, сообщения, жирный текст),
который легко преобразуется в HTML и HTML-код. Используются для создания документации, позволяя состедоточиться на тексте, а не на сложном коде.

---
Основные возможности Markdown - это быстрое и простое форматирование текста (заголовки, списки, ссылки,
выделение), сохраняющее читабельность, для последующего преобразования в формат HTML или другие форматы.Идеально для документации, заметок, README-файлов, блога и сообщений с синтаксисом. Использование
обычных символов вместо сложного кода,  делает его удобным и интуитивно понятным.
