## Лабораторная работа №3. Работа с Markdown-разметкой и базовое использование LaTeX в документации проекта

#### Markdown («маркдаун») — облегчённый язык разметки для форматирования текста. В отличие от сложных языков разметки, таких как HTML или XML, Markdown использует простые знаки препинания.
#### Markdown находит широкое применение в различных сферах:
- Веб-разработка
- Блоги и платформы для написания
- Электронные письма и форумы
- Статические сайты
- Документация
- Системы управления версиями
- Примечания и заметки
- Презентации

#### Markdown нужен разработчикам для форматирования текста, делая его читабельным как в исходном виде, так и в преобразованном формате, например в HTML. В отличие от HTML с его закрывающими и открывающими тегами, Markdown использует интуитивно понятные символы: звёздочки для выделения, решётки для заголовков и дефисы для списков.
